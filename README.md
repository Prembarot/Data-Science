# 📊 Data Science Foundations with NumPy, Pandas & Matplotlib
This repository contains my structured practice of essential Data Science libraries used for data analysis, data manipulation, and data visualization.
It includes hands-on programs and experiments using powerful Python libraries that form the foundation of Machine Learning and Analytics workflows.
---
## 📘 Libraries Covered
### 🔹 NumPy (Numerical Computing)
* Array Creation and Indexing
* Mathematical and Statistical Operations
* Reshaping and Broadcasting
* Linear Algebra Basics
* Random Number Generation
### 🔹 Pandas (Data Analysis & Manipulation)
* Series and DataFrame Creation
* Data Cleaning and Preprocessing
* Handling Missing Values
* Filtering and Sorting Data
* GroupBy Operations
* Merging and Joining DataFrames
* Reading/Writing CSV Files
### 🔹 Matplotlib (Data Visualization)
* Line Charts
* Bar Graphs
* Histograms
* Pie Charts
* Scatter Plots
* Customizing Plots (labels, titles, legends, grids)
---

To develop practical skills required for:
* Data Analysis

* Machine Learning Preprocessing
* Exploratory Data Analysis (EDA)
* Real-world Dataset Handling
